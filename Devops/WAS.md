# WAS(Web Application Server)
동적인 처리를 담당하는 서버로, 기본적으로 웹서버(WebServer)의 기능도 모두 지원한다.
Java에서 주로 사용하는 최신 버전 톰캣의 경우 Apache Tomcat으로, WebServer인 Apache까지 포함한 개념이라고 볼수있다.<br>
ex. Tomcat, Jeus 등등
